# 合同大师 (Contract Master)

[English](./README.md) | [简体中文](./README_zh.md)

“合同大师”是一款先进的合同管理中间件，旨在连接 SaaS CRM 系统（如 Salesforce、HubSpot、钉钉）与企业内部业务系统。它为合同提供了统一的数据模型，并通过智能规则引擎和 AI 驱动的见解，增强了合规风险检测和数据的完整性。

## 项目概览

本系统作为合同数据的中心枢纽，通过以下方式确保标准化和准确性：
- **数据集成**：支持从多个 CRM 平台实时同步，并具备向下游系统主动推送数据的能力。
- **规则引擎**：灵活的可视化逻辑引擎，支持 LOGIC、GROOVY 和 REGEX，用于合规性与风险验证。
- **AI 智能化**：使用大语言模型（如硅基流动、OpenAI、DeepSeek）自动解析 PDF/Docx 文档，补充合同元数据。
- **身份与访问控制 (RBAC)**：完善的角色访问控制，支持多租户隔离及精细化的数据范围控制（全局/部门/个人）。

## 技术栈

### 后端 (Backend)
- **核心框架**：Java 17, Spring Boot 3.2.x
- **持久化**：Spring Data JPA (Hibernate 6.x), H2/PostgreSQL
- **规则引擎**：自定义脚本引擎 (Groovy/Regex)
- **人工智能**：通过 OpenAI 兼容协议集成主流 LLM 服务商
- **文档处理**：Apache PDFBox 用于内容提取
- **消息中间件**：Spring Kafka 用于异步评估任务

### 前端 (Frontend)
- **核心框架**：Vue 3 (Composition API)
- **UI 组件库**：Element Plus
- **构建工具**：Vite
- **国际化**：Vue I18n (全系统支持中英文切换)
- **状态管理**：Pinia

## 项目结构

```text
contract-master/
├── backend/          # Spring Boot 应用（基于 DDD 架构设计）
├── frontend/         # Vue 3 + Vite 前端应用

├── scripts/          # 自动化脚本（开发环境启动、环境检查、E2E）
└── docs/             # 项目文档与规范
```

## 核心特性

- **AI 智能提取**：自动从上传的 PDF/Docx 文档中提取 5 个以上关键字段，并提供视觉高亮以供人工核对。
- **统一合规驾驶舱**：集中的风险检测仪表盘，支持实时告警及问题全生命周期跟踪。
- **动态权限矩阵**：按角色配置模块访问权限和数据可见范围，配置即时生效。
- **集成中心**：定义出站 Webhooks 和字段映射规则，将已验证的数据无缝分发到下游 ERP/CRM 系统。
- **审计追踪**：完整的系统变更历史和用户活动记录，满足合规审计需求。

## 快速上手

### 环境准备
- **Java 17** 或更高版本
- **Node.js** (v18+)
- **Docker** (用于基础设施服务)

### 一键开发环境启动
运行开发环境脚本（包含自动依赖检查）：
- **Unix/macOS**: `./scripts/dev.sh`
- **Windows**: `scripts\dev.bat`
